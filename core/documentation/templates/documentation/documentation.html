AUTHENTICATION

This API uses (knox) token authentication (Django Rest Knox).
This means that all requests made to essential endpoints must provide the token inside the header config.

BASE URL

http://188.166.43.176/

HEADER

Authorization: Token {token}
Content-Type: application/json

TOKENS

A token is generated anytime a user logs in or signs up.
Collect this token and parse it to the header config when making a call to essential endpoints.
